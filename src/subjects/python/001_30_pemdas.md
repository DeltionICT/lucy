Python Operators Precedence Rule – PEMDAS  
- P – Parentheses. = ()
- E – Exponentiation. = **
- M – Multiplication. = *
- D – Division. = /
- A – Addition. = +
- S – Subtraction. = -

```python
x = 1
```
Output
> De uitkomst is: 8

```python
x = 2
```
Output
> De uitkomst is: 45

```python
x = 0
```
Output
> De uitkomst is: 1
